# Mess — мессенджер

Мессенджер Mess: React (фронтенд), Node.js + Express (бэкенд), SQLite через встроенный `node:sqlite` (база данных), Socket.io (сообщения в реальном времени).

Требуется Node.js 22+.

## Возможности

- Регистрация и вход
- Личные чаты между пользователями
- Поиск пользователей и создание нового чата
- Сообщения в реальном времени
- Индикатор «печатает…»
- Тёмная тема интерфейса

## Запуск

```bash
# Установка зависимостей
npm run install:all

# Запуск сервера и клиента (старый процесс на порту 3001 останавливается автоматически)
npm run dev
```

Если видите ошибки «Сервер устарел» или «Ошибка загрузки» — остановите все окна терминала и снова выполните `npm run dev`.
Проверка: http://localhost:3001/api/health должен вернуть `{"ok":true,"version":2}`.

- Клиент: http://localhost:5173
- API: http://localhost:3001

Для проверки откройте два окна браузера (или обычное + инкогнито), зарегистрируйте двух пользователей и начните переписку.

## Структура

```
mess/
├── server/          # Node.js API + WebSocket
│   └── src/
│       ├── db.js    # SQLite схема
│       ├── routes/
│       └── socket.js
└── client/          # React (Vite)
    └── src/
```

База данных создаётся автоматически в `server/data/mess.db`.
